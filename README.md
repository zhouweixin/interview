## 4.Map(HashMap, LinkedHashMap, TreeMap)对比：MyMap.java

    Map操作：put的顺序
    abc=D
    1=A

    1=A
    2=B

    3=C
    LinkedHashMap: key有序(put的顺序)
    abc=D
    1=A

    3=C
    TreeMap: key有序(默认升序)
    1=A
    2=B
    3=C

    abc=D
    3=C

    1=A
